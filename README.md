Папка /HomeTask1:
Описания приложения.
Приложение состоит из двух экранов в навигейшен стеке. Модули приложения выполнены на основе архитектурного паттерна MVC. Верстка осуществлена программно, без сторибордов.
На первом экране, по умолчанию, отображается список всех автомобилей с помощью UITableView. По умолчанию добавлены четыре автомобиля. На экране, над списком имеется возможность отсортировать автомобили по типу кузова с помощью UIPickerView встроенного в UITextField. Внизу экрана имеется кнопка для добавления нового автомобиля.
При нажатии на кнопку добавления нового автомобиля появляется второй экран приложения на котором отображается UIStackView с полями для ввода параметров автомобиля. Год выпуска и номер автомобиля являются необязательными полями для ввода. При нажатии кнопки Done в UINavigationBar и отсутствии ввода наименования производителя или модели отображается UIAlertController с оповещением о некорректности введенных данных. Также при попытке ввести значение не типа Int в поле года производства автомобиля отображается оповещение с просьбой ввести номер цифрами. 
При корректном вводе параметров автомобиля и дальнейшем нажатии на кнопку Done, происходит переход на корневой вьюконтроллер. Где в списке отображаются все автомобили, включая добавленный.
Приложение выполнено с минимумом функционала UIKit, в соответствии с техническим заданием.
